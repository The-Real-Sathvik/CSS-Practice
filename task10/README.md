Create a 3x3 Grid Layout

Task: Create a 3x3 grid layout where each cell contains a numbered box. The grid should be responsive, and the boxes should adjust their size based on the container's width.

Instructions:
-> Create a container div with the class name grid-container.
-> Inside the container, create nine div elements with the class name grid-item.
-> Use CSS Grid to arrange the grid-item elements into a 3x3 grid.
-> Style each grid-item with a background color, padding, and centered text.
-> Ensure the grid is responsive so that when the container's width decreases, the grid items maintain their aspect ratio.
